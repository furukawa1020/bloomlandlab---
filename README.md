## 🛠️ 技術スタック

- **Framework**: Next.js 14 (App Router, SSG)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Animation**: Framer Motion
- **Forms**: Netlify Forms
- **Share**: react-share
- **Deployment**: Netlify
- **SEO**: 完全対応（JSON-LD, OpenGraph, sitemap.xml）

## 🚀 セットアップ

```bash
# 依存関係インストール
npm install

# 開発サーバー起動
npm run dev

# ビルド
npm run build

# 静的ファイル出力（Netlify用）
npm run export
```
